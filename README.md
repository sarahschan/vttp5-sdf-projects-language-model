# SDF Projects - Language Model
- Program reads a single .txt file and loads the writing into a language model
- Language model shows a first word, and the number of next word occurences
- User may generate a sentence/phrase using this model
- User selects the word to start, number of words in the sentence/phrase, and hallucination level (0-1)

## Usage
- Need to compile classes / create .jar file first

java -cp classes Main data/[fileName]
